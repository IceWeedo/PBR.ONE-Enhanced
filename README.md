# PBR.ONE-Enhanced

**Description:**  
PBR.ONE-Enhanced makes PBR.ONE easier and nicer to use. It adds a visual hub and material catalogue. Optional textures (AO, Displacement, Metalness) are handled automatically.

**Features:**  
- Material catalogue with images (`material-catalogue.html`)  
- Detects optional textures automatically  
- Simple JSON file for adding new materials  
- Works with `.png` textures and full HTTP URLs  
- Easy to add new modules in `/modules/`  

**Installation:**  
1. Download or clone the repo  
2. Put files on a web server  
3. Open `index.html` in your browser  

**Future:**  
- Add more modules in `/modules/`  
- Improved previews, filters, or export tools
